# Letter Boxed game
## Getting Started
### Dependencies

* [Deno](https://deno.land/)

### Installing

1. You need first to install deno
```
> brew install deno
```

### Executing program

* Execute the program
```
> deno run --allow-read  --allow-write generate.ts
```
The generated puzzle will appear in `game.json`

### Testing

* Execute
```
> deno test --allow-read  generate.spec.ts
```
